🤖 Flex Sensor Controlled Servo Motor

A simple Arduino-based project that controls a servo motor using a flex sensor. When the flex sensor bends, the servo motor rotates according to the sensor value.


---


🚀 Features

Real-time servo motor control

Flex sensor input processing

Smooth and responsive movement

Beginner-friendly Arduino project



---

🛠️ Components Used

Arduino UNO

Flex Sensor

Servo Motor

Breadboard

Jumper Wires



---

⚙️ How It Works

The flex sensor changes its resistance when bent. Arduino reads the analog value from the sensor and maps it into a servo angle between 0° and 180°. The servo motor then rotates according to the sensor movement.
